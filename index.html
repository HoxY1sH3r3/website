Alright let's go

Legendary Samurai is an rng bot for the sole purpose of collecting stuff, gaining coins and more!

Main part!

/RNG

This is our main command that allows you to roll for an item like a gacha system with rarities like common, uncommon, rare, legendary, mythic and more will arrive soon!

/Inventory

This command allows a user to find out what the currently have and is a necesarity along with /rng to even make this bot a gacha bot.

/Flex

This command is so that you can show others what cards you have because its just cool.

/Gamble

I am a person who loves gambling but hates losing REAL money so i built this command for others who are like me.
